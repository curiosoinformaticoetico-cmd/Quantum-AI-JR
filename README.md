# Quantum AI JR

Proyecto enfocado en el desarrollo de Inteligencia Artificial integrada con computación cuántica.

## 🎯 Visión
Construir sistemas inteligentes que permitan:
- Combatir la corrupción
- Optimizar la justicia
- Crear gobiernos eficientes y transparentes

## ⚙️ Tecnologías
- Python
- Qiskit (simulación cuántica)
- Machine Learning

## 🚀 Objetivo
Desarrollar una arquitectura híbrida entre IA clásica y computación cuántica que evolucione con el avance tecnológico.

## 📌 Estado del proyecto
Fase inicial - simulación y pruebas
